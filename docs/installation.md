# Installation

## CURL SSL backend selection

AmiSSL: --with-amissl
BearSSL: --with-bearssl
GnuTLS: --with-gnutls.
mbedTLS: --with-mbedtls
NSS: --with-nss
OpenSSL: --with-openssl (also for BoringSSL and libressl)
rustls: --with-rustls
Schannel: --with-schannel
Secure Transport: --with-secure-transport
wolfSSL: --with-wolfssl
